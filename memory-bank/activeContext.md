## Корисні команди (канонічні, ручне управління)

### Apache (httpd.service) - ручне управління
⚠️ **ВАЖЛИВО**: Використовувати ТІЛЬКИ ці команди. Автозапуск відключено.
- **Старт**: `sudo systemctl start httpd`
- **Стоп**: `sudo systemctl stop httpd`
- **Перезапуск**: `sudo systemctl restart httpd`
- **Статус**: `sudo systemctl status httpd`
- **Відключити автозапуск** (якщо включений): `sudo systemctl disable httpd`

### Laravel Maintenance
- **Включити заглушку**: `php artisan down --secret="maintenance2244"`
- **Виключити заглушку**: `php artisan up`

### Команди проекту
- **Очистка кешів**: `php artisan ops:refresh`
- **Очистка + перезапуск Apache**: `php artisan ops:refresh --restart-httpd`
- **Повний рестарт системи**: `php artisan ops:restart`
- **Діагностика**: `php artisan ops:doctor`
