# Product Context - GLF Bikube

## Чому проект існує
GLF Bikube вирішує проблему централізації та управління різноманітними послугами доставки в місті Нарвік. Система об'єднує різні категорії послуг (доставка їжі, товарів, майстерні послуги, евакуація тощо) в єдину платформу з інтелектуальною маршрутизацією та оптимізацією.

## Проблеми, які вирішує система
1. **Фрагментація послуг**: Різні провайдери послуг працюють окремо
2. **Неефективна маршрутизація**: Відсутність оптимізації маршрутів
3. **Складність інтеграції**: Партнери потребують простого API
4. **Відсутність аналітики**: Немає централізованого аналізу операцій
5. **Проблеми з SLA**: Складність відстеження та виконання SLA

## Як система повинна працювати
1. **Публічна вітрина**: Клієнти переглядають каталог послуг, обирають та оформляють замовлення
2. **API для партнерів**: Партнери інтегруються через OAuth2/OIDC та отримують webhooks
3. **Адмін-панель**: Адміністратори управляють системою через Filament
4. **Мобільний додаток**: Виконавці отримують задачі та виконують їх через PWA
5. **Маршрутизація**: Система автоматично оптимізує маршрути та розраховує ETA
6. **Платежі**: Інтеграція з Stripe та Vipps для безпечних платежів
7. **Аналітика**: Dashboard з метриками, SLA, ефективністю

## Цілі користувацького досвіду
- **Швидкість**: Швидке оформлення замовлення (< 2 хвилин)
- **Прозорість**: Реальний час відстеження замовлення
- **Надійність**: Гарантоване виконання SLA
- **Зручність**: Інтуїтивний інтерфейс для всіх типів користувачів
- **Мобільність**: Повна функціональність на мобільних пристроях

## Унікальні особливості
- ML-прогнози спросу та ETA
- Динамічне ціноутворення (surge pricing)
- A/B тестування цін
- Телематика (GPS/OBD події)
- Геозамки (geofences)
- Персоналізація вітрини (рекомендації)
- Carbon/ESG звітність

## Структура услуг: Категории и подкатегории

Представленный план услуг структурирован по восьми основным категориям. Все дополнительные опции (включая сезонные) перенесены в отдельный раздел «Дополнительные опции».

### 1. Логистика и доставка (Delivery & Logistics)

Центральная категория, объединяющая все виды транспортировки товаров, от продуктов питания до крупногабаритных грузов.

| Категория | Подкатегория | Описание услуги | Магазины и особенности |
| :---- | :---- | :---- | :---- |
| **Доставка продуктов и товаров (Retail Delivery)** | **Personal Shopping** | Сборка по списку, замены по звонку/чату. | **Продукты:** Rema 1000, Kiwi, Coop Extra, Coop Prix, Joker, Bunnpris, Spar, Meny, Eurospar. **Прочее:** Local bakeries & delis. |
|  | **Click & Collect** | Забор предоплаченных заказов из магазинов. | Rema 1000, Kiwi, Clas Ohlson, Elkjøp и др. |
|  | **Холодная цепь** | Использование термосумок и холодо-аккумуляторов (для молочки, мяса, заморозки). | Любые продуктовые сети. |
|  | **Доставка еды и напитков** | Доставка из кафе, ресторанов и фастфудов (партнерские рестораны вне Wolt). | B2B: Кейтеринг-сеты, кофе-маршруты для офисов. |
| **Специализированная доставка (Specialized Handling)** | **Крупногабарит / Тяжёлое** | Перевозка тяжелых и больших предметов: «2 курьера», подъем по этажам, размещение в комнате (**Room-of-Choice**). | **Мебель/Техника:** Elkjøp, Power, JYSK, IKEA (групповая доставка). |
|  | **White-Glove** | Премиум-доставка: занос "до комнаты", распаковка, вынос упаковки. | **Спецтовары:** High-end электроника, специализированное оборудование. |
|  | **B2B Last-Mile** | Ежедневные выезды по слотам для локальных магазинов. | Biltema, Clas Ohlson, Obs Bygg, Montér, Byggmax. |
| **Обратная логистика (Reverse Logistics)** | **Возвраты** | Забор возвратов и доставка их в магазин/почту. | **B2B-подписка:** Сбор возвратов онлайн-магазинов. |
| **Трансграничная доставка (Cross-Border)** | **Швеция / Riksgränsen** | Доставка по графику 2–3 раза в неделю (40–45 минут от Нарвика). | ICA Nära Riksgränsen. Требует учета таможенных правил для коммерческих грузов. |

### 2. Переезд и транспортировка (Moving & Transport)

Комплексные услуги, связанные с изменением места жительства или офиса.

| Категория | Подкатегория | Описание услуги | Особенности |
| :---- | :---- | :---- | :---- |
| **Переезд «под ключ» (Full-Service)** | **Пакеты S/M/L** | Фиксированные пакеты в зависимости от объема (кол-во коробок, 1–2 курьера, прицеп). | Время работы ограничено (например, 1 апреля – 1 ноября для фиксированных цен). |
|  | **Подготовка и сборка** | Упаковка, разборка/сборка мебели (связь с «УСЛУГИ МАСТЕРА»), защитные материалы, фото-опись. | В Нарвике почасовая стоимость: 950–1750 крон за 1–2 человека с автомобилем. |
|  | **Утилизация после переезда** | Вывоз мусора и старой мебели (связь с «ЭКО услуги»). |  |
| **Эвакуатор / Помощь на дорогах (Roadside Assistance)** | **Экстренная помощь** | Прикуривание, подвоз топлива, замена колеса. | Услуги часто покрываются страховкой (Kasko/Delkasko) с франшизой (750 крон). |
|  | **Транспортировка** | Буксировка и транспортировка на СТО. | Требует наличия лицензии (*Løyve*). |

### 3. Курьерские и индивидуальные поручения (Personal & Express Tasks)

Услуги, связанные с быстрой передачей документов или выполнением индивидуальных задач.

| Категория | Подкатегория | Описание услуги | Особенности |
| :---- | :---- | :---- | :---- |
| **Индивидуальные поручения** | **Документы и бюрократия** | Скан/печать/подпись/завезти документы, забрать талон или отстоять очередь. |  |
|  | **Покупка и доставка** | Купить и привезти товары, не связанные с продуктами питания. |  |
| **Экспресс-услуги** | **Экспресс-слоты** | Доставка 60/120 мин в черте Нарвика. |  |

### 4. Услуги мастера и ремонт (Handyman & Repair)

Технические услуги, требующие определенных навыков (без привлечения сертифицированных бригад по спецработам).

| Категория | Подкатегория | Описание услуги | Магазины и особенности |
| :---- | :---- | :---- | :---- |
| **Мелкий бытовой ремонт** | **Сборка и установка** | Сборка мебели (JYSK, Skeidar), подключение бытовой техники (стиралка, посудомойка). | **Источники материалов:** Clas Ohlson, Biltema, Jula. |
|  | **Монтаж** | Монтаж ТВ/карнизов, мелкие работы по электрике/сантехнике. |  |
| **Строительные услуги** | **Бригада** | Пол, покраска, гипсокартонные работы (почасово или по смете). | Для спецработ необходимы ИП/фирмы-партнеры с соблюдением требований HSE. |
|  | **Закупка материалов** | Доставка стройматериалов (Montér, Byggmax, Obs Bygg). |  |

### 5. Социальная помощь (Elderly & Disability Support)

Услуги, ориентированные на помощь пожилым людям и лицам с ограниченными возможностями, включая транспортное сопровождение.

| Категория | Подкатегория | Описание услуги | Особенности и требования |
| :---- | :---- | :---- | :---- |
| **Абонементное обслуживание** | **"Omsorg"** | 1–2 визита в неделю (продукты, мелкие бытовые задачи, записи к врачу/NAV). | Муниципалитеты несут ответственность за услуги здравоохранения и ухода. |
|  | **Wellness-check** | Регулярная проверка безопасности и благополучия. |  |
| **Сопровождение и транспорт** | **Транспортное сопровождение (Shuttle)** | Перевозка и сопровождение пожилых/инвалидов в учреждения, к врачу, в NAV. | Требуется наличие **Kjøreseddel** (профессиональный сертификат) для водителя. |
|  | **Техническая помощь** | Установка простых бытовых помощников (датчики, поручни) — через партнеров. |  |

### 8. Дополнительные опции и B2B (Value-Added & Corporate Services)

Общие функции, применимые ко всем категориям для увеличения маржи и удобства.

| Опция | Описание | Применение |
| :---- | :---- | :---- |

| **Управление рисками** | Дополнительная страховка груза, фото/подпись при доставке. | Применимо ко всем услугам. |
| **B2B и лояльность** | Корпоративные счета (B2B), отчеты и НДС-квитанции, купоны/промокоды. | Применимо ко всем услугам. |
