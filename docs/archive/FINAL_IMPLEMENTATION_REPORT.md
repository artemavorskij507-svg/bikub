## 🗄️ Структура базы данных

### Основные таблицы:
- `service_categories` - Категории услуг (Care, Eco, Market, etc.)
- `service_types` - Типы услуг с ценами и навыками
- `pricing_rules` - Правила ценообразования
- `schedule_slots` - Временные слоты (утро/день/вечер)
- `orders` - Заказы с привязкой к слотам
- `order_items` - Позиции заказов
- `tasks` - Задачи для исполнителей
- `roles` - Роли пользователей
- `user_roles` - Связь пользователей и ролей
- `payment_settings` - Настройки Stripe

## 🎯 Доступные URL

### Публичные страницы:
- `http://glfbikube.local/` - Главная каталог
- `http://glfbikube.local/catalog` - Каталог услуг
- `http://glfbikube.local/care` - Услуги Care
- `http://glfbikube.local/eco` - Услуги Eco
- `http://glfbikube.local/market` - Услуги Market
- `http://glfbikube.local/order/{service_code}` - Форма заказа

### Админ панель:
- `http://glfbikube.local/admin` - Дашборд с виджетом платежей
- `http://glfbikube.local/admin/service-types` - Управление услугами
- `http://glfbikube.local/admin/pricing-rules` - Правила ценообразования
- `http://glfbikube.local/admin/schedule-slots` - Временные слоты
- `http://glfbikube.local/admin/payment-settings` - Настройки Stripe

### API endpoints:
- `GET /api/v1/service-types` - Список услуг
- `POST /api/v1/orders` - Создание заказа
- `POST /api/v1/orders/{id}/payment/intent` - Создание платежа
- `POST /api/v1/orders/{id}/payment/confirm` - Подтверждение платежа
