# Roadside & Tow — Чек-лист для QA

## 📋 Общая информация

Этот чек-лист предназначен для ручного тестирования модуля Roadside & Tow в админ-панели и Worker LK.

---

## 🚀 Подготовка к тестированию

### 1. Заполнение демо-данными

- [ ] Запустить команду `php artisan roadside:seed-demo` (или использовать кнопку на странице Debug & Demo)
- [ ] Проверить, что команда выполнилась без ошибок
- [ ] Убедиться, что созданы:
  - [ ] Партнёры-эвакуаторы (минимум 3-5)
  - [ ] Дорожные помощники (минимум 5-8)
  - [ ] Пресеты Roadside (минимум 5-7)
  - [ ] Пресеты осмотра авто (минимум 3-4)
  - [ ] Экстренные вызовы (опционально, 5 штук)
  - [ ] Заявки на осмотр (опционально, 3 штуки)

---

## 🔍 Проверка админ-панели

### 2. Навигация и доступ

- [ ] В левом меню админ-панели присутствует раздел "Roadside & Tow"
- [ ] В разделе отображаются все пункты меню в правильном порядке:
  1. Roadside Dashboard
  2. Экстренные вызовы
  3. Roadside Dispatch
  4. Виды работ Roadside
  5. Дорожные помощники
  6. Партнёры-эвакуаторы
  7. Заявки на осмотр авто
  8. Пресеты осмотра авто
  9. Debug & Demo

### 3. Партнёры-эвакуаторы

- [ ] Открыть страницу "Партнёры-эвакуаторы"
- [ ] Проверить, что отображаются созданные партнёры
- [ ] Проверить фильтры:
  - [ ] Фильтр по типу (towing_service, roadside_mobile, etc.)
  - [ ] Фильтр по гео-зоне
  - [ ] Фильтр по активности
- [ ] Создать нового партнёра:
  - [ ] Заполнить обязательные поля (имя, тип, телефон)
  - [ ] Выбрать гео-зону
  - [ ] Указать capabilities (возможности)
  - [ ] Установить приоритет
  - [ ] Сохранить
- [ ] Проверить, что партнёр появился в списке
- [ ] Отредактировать партнёра
- [ ] Удалить тестового партнёра (если нужно)

### 4. Дорожные помощники

- [ ] Открыть страницу "Дорожные помощники"
- [ ] Проверить, что отображаются созданные помощники
- [ ] Создать нового помощника:
  - [ ] Привязать к пользователю (или создать нового)
  - [ ] Указать тип и модель транспорта
  - [ ] Указать навыки (skills)
  - [ ] Указать оборудование (equipment)
  - [ ] Установить статус
  - [ ] Сохранить
- [ ] Проверить, что помощник появился в списке
- [ ] Отредактировать помощника

### 5. Виды работ Roadside

- [ ] Открыть страницу "Виды работ Roadside"
- [ ] Проверить, что отображаются созданные пресеты
- [ ] Создать новый пресет:
  - [ ] Указать код (code) и название (label)
  - [ ] Выбрать тип услуги (service_type)
  - [ ] Указать базовую цену
  - [ ] Указать, требуется ли партнёр
  - [ ] Сохранить
- [ ] Проверить, что пресет появился в списке
- [ ] Отредактировать пресет

### 6. Пресеты осмотра авто

- [ ] Открыть страницу "Пресеты осмотра авто"
- [ ] Проверить, что отображаются созданные пресеты
- [ ] Создать новый пресет:
  - [ ] Указать название и описание
  - [ ] Указать цену
  - [ ] Добавить чек-лист (checklist)
  - [ ] Сохранить
- [ ] Проверить, что пресет появился в списке
- [ ] Отредактировать пресет

### 7. Экстренные вызовы

- [ ] Открыть страницу "Экстренные вызовы"
- [ ] Проверить, что отображаются созданные вызовы (если есть)
- [ ] Создать новый экстренный вызов:
  - [ ] Выбрать клиента
  - [ ] Указать тип инцидента
  - [ ] Указать описание
  - [ ] Указать координаты (lat, lng)
  - [ ] Назначить помощника (опционально)
  - [ ] Назначить партнёра (опционально)
  - [ ] Сохранить
- [ ] Проверить, что вызов появился в списке
- [ ] Изменить статус вызова
- [ ] Отредактировать вызов

### 8. Заявки на осмотр авто

- [ ] Открыть страницу "Заявки на осмотр авто"
- [ ] Проверить, что отображаются созданные заявки (если есть)
- [ ] Создать новую заявку:
  - [ ] Выбрать клиента
  - [ ] Выбрать пресет осмотра
  - [ ] Указать данные об автомобиле
  - [ ] Указать адрес
  - [ ] Назначить помощника (опционально)
  - [ ] Сохранить
- [ ] Проверить, что заявка появилась в списке
- [ ] Изменить статус заявки
- [ ] Отредактировать заявку

### 9. Roadside Dashboard

- [ ] Открыть страницу "Roadside Dashboard"
- [ ] Проверить, что отображаются активные заказы Roadside
- [ ] Проверить виджеты статистики (если есть)
- [ ] Проверить фильтры и поиск
- [ ] Проверить действия с заказами (назначение исполнителя, изменение статуса)

### 10. Roadside Dispatch

- [ ] Открыть страницу "Roadside Dispatch"
- [ ] Проверить, что отображаются активные заказы
- [ ] Выбрать заказ
- [ ] Назначить исполнителя
- [ ] Назначить партнёра
- [ ] Изменить статус заказа
- [ ] Проверить фильтры (статус, тип услуги, зона)

### 11. Debug & Demo

- [ ] Открыть страницу "Debug & Demo"
- [ ] Проверить, что отображается статистика по модулю
- [ ] Проверить, что отображаются последние экстренные вызовы
- [ ] Проверить, что отображаются последние заявки на осмотр
- [ ] Нажать кнопку "Заполнить демо-данными"
- [ ] Проверить, что появилось уведомление об успешном выполнении
- [ ] Обновить страницу и проверить, что статистика обновилась

## 🔗 Интеграция с заказами

### 12. Связь Roadside-партнёра с заказом

- [ ] Открыть заказ, связанный с Roadside-услугой
- [ ] В форме редактирования заказа найти секцию "Roadside / Партнёр"
- [ ] Выбрать партнёра из списка
- [ ] Заполнить дополнительные данные партнёра (metadata)
- [ ] Сохранить заказ
- [ ] Проверить, что партнёр отображается в таблице заказов
- [ ] Проверить, что партнёр отображается в Worker LK

## 👷 Worker LK

### 13. Отображение Roadside-партнёра в Worker LK

- [ ] Войти в Worker LK
- [ ] Открыть заказ, к которому привязан Roadside-партнёр
- [ ] Проверить, что отображается блок с информацией о партнёре:
  - [ ] Название партнёра
  - [ ] Телефон (с возможностью звонка)
  - [ ] Гео-зона
  - [ ] Возможности (capabilities)
