# Тестовые сценарии GLF BiKube

## Дата: 2025-11-19

### Сценарий 1: Создание заказа доставки продуктов

1. Залогиниться как клиент (demo@glf.no / password)
2. Перейти на главную страницу `/`
3. Нажать "Заказать доставку продуктов"
4. Выбрать магазин (Rema 1000, Kiwi и т.д.)
5. Добавить товары в корзину
6. Заполнить адрес доставки
7. Подтвердить заказ
8. Проверить в `/account/orders` - заказ должен быть виден
9. Открыть детали заказа - должны отображаться адреса, ETA, статус

### Сценарий 2: Создание заказа готовой еды

1. Залогиниться как клиент
2. Перейти на главную страницу
3. Выбрать ресторан из блока "Популярные рестораны"
4. Добавить блюда
5. Заполнить адрес доставки
6. Подтвердить заказ
7. Проверить в `/account/orders`
8. Открыть детали - должен быть виден трекинг

### Сценарий 3: Создание заказа крупногабарита

1. Залогиниться как клиент
2. Перейти на главную страницу
3. Нажать "Доставка крупногабарита"
4. Заполнить размеры, вес, адреса
5. Подтвердить заказ
6. Проверить в `/account/orders`
7. Открыть детали - должны быть видны размеры и адреса

### Сценарий 4: Вызов мастера

1. Залогиниться как клиент
2. Перейти в раздел "Мастер на час"
3. Выбрать услугу (ремонт, сборка и т.д.)
4. Заполнить адрес и время
5. Подтвердить заказ
6. Проверить в `/account/orders`
7. Открыть детали - должна быть видна информация о мастере

### Сценарий 5: Эко-вывоз

1. Залогиниться как клиент
2. Перейти в раздел "Эко-услуги"
3. Выбрать тип вывоза (мебель, техника и т.д.)
4. Заполнить адрес и описание
5. Подтвердить заказ
6. Проверить в `/account/orders`
7. Открыть детали - должна быть видна информация об утилизации

### Сценарий 6: Социальное посещение

1. Залогиниться как клиент
2. Перейти в раздел "Социальная помощь"
3. Выбрать тип визита
4. Заполнить данные клиента и время
5. Подтвердить заказ
6. Проверить в `/account/orders`
7. Открыть детали - должна быть видна информация о помощнике

### Сценарий 7: Индивидуальное поручение

1. Залогиниться как клиент
2. Перейти в раздел "Индивидуальные поручения"
3. Заполнить описание, адреса
4. Указать срочность
5. Подтвердить заказ
6. Проверить в `/account/orders`
7. Открыть детали - должна быть видна информация о поручении

### Сценарий 8: Админ-панель

1. Залогиниться как админ (keks@glf.no)
2. Перейти на `/admin`
3. Проверить доступность всех ресурсов:
   - Delivery Orders
   - Orders
   - Users
   - Stores
   - Restaurants
   - Handyman Services
   - Eco Orders
   - Social Care Orders
   - Errand Tasks
4. Открыть Delivery Operations Board
5. Проверить фильтры и действия

### Сценарий 9: Главная страница

1. Открыть главную страницу `/` (без авторизации)
2. Проверить отображение:
   - Категории услуг (7-8 блоков)
   - Популярные магазины (8 магазинов)
   - Популярные рестораны (8 ресторанов)
   - Популярные услуги мастера
   - Эко-услуги
   - Активные доставки
   - Индивидуальные поручения
3. Все ссылки должны вести на реальные страницы
4. Нет заглушек и placeholder'ов

### Сценарий 10: Производительность

1. Очистить кеш: `php artisan optimize:clear`
2. Открыть главную страницу - должно загрузиться быстро (< 500ms)
3. Открыть `/account/orders` - должно загрузиться быстро
4. Открыть `/admin` - должно загрузиться быстро
5. Проверить кеширование: повторные запросы должны быть быстрее
